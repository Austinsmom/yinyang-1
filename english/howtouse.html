How To Use

You can download milk.coffee from GitHub. A typical file layout would be like below (...under construction)

  • DOCUMENT_ROOT
    • index.html
    • about.html
    • contact.html
    • style
      • template.html
      • style.css
    • js
      • milk.js
      • jquery-1.7.1.min.js

index.html、about.html、contact.html are document HTML files. style/template.html is a HTML file for layout.

You need insert a few rows of code to HEAD tag in index.html (etc).

<link rel="template" href="style/template.html">
<script src="js/milk.js"></script>

  • Don't use such layout tags: DIV, SECTION, ARTICLE
  • Headings(H1,H2,H3,H4), paragraphs(P)、images(IMG)、lists(UL,OL,LI)、link(A) are good to use.
  • Don't include information if it can be loaded by Ajax or hSQL.
  • Use META tags for your copyright, keywords, summary and categories etc.
